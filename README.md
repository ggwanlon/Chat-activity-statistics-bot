# Chat-activity-statistics-bot
这是一个 tg 机器人用于统计群内成员活跃度
1. 积分规则

用户在群内发送有效消息时，增加 1 活跃积分
有效消息定义：
非机器人消息
非系统消息
文本长度不少于 5 个字符
60 秒内重复发言仅记 1 次
每个用户每日最多获得 50 分
所有积分按群维度独立统计

2. 榜单规则
每天北京时间 09:00 自动发送昨日活跃榜 Top 10
榜单按昨日新增积分降序排列
同时发送当前周期总分榜 Top 5
同分时按最后活跃时间排序

3. 周期规则
采用全群统一 30 天积分周期
周期结束时自动发送周期结算榜
结算完成后所有用户周期积分清零
保留历史归档记录

4. 管理规则
管理员通过私聊机器人登录
登录成功后获得临时管理权限
支持查看、加分、扣分、重置、查询榜单、查看日志
所有后台操作均记录审计日志

5. 安全规则
管理密码哈希存储
登录失败次数限制
管理会话自动过期
敏感操作记录日志
可限制仅指定 Telegram 用户 ID 可登录管理后台
